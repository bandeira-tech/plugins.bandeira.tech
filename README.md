# plugins.bandeira.tech

A Claude Code marketplace for [BANDEIRA·TECH](https://github.com/bandeira-tech) plugins.

## Install

```
/plugin marketplace add bandeira-tech/plugins.bandeira.tech
```

Then `/plugin install <name>` for any plugin below.

## Plugins

| Plugin | Description |
|---|---|
| [`b3nd`](./b3nd/) | Build with B3nd inside Claude Code — own your data, ship apps as URI shapes + on-the-fly UI. Hosts your rig as MCP via the `bnd` CLI. |
